## Funcionalidade

 O Jogo da Adivinhação é um jogo simples e divertido, onde o jogador deve adivinhar um número gerado aleatoriamente pelo sistema dentro de um intervalo definido. O objetivo é acertar o número com o menor número de tentativas possíveis.

## Como Jogar

- O sistema gera um número aleatório dentro de um intervalo predefinido (por exemplo, entre 1 e 20).

- O jogador insere um palpite.

- O jogo informa se o número correto é maior ou menor do que o palpite do jogador.

- O jogador continua tentando até acertar o número.

- Quando o jogador acerta, o jogo informa o número de tentativas realizadas.

## Requisitos

- NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.

## Como Utilizar o Jogo da Advinhação

Clone o Repositório

```
git clone https://github.com/ovatsuGMattos/JogoAdvinhacao
```
Navegue ate a pasta Raiz da solução
```
cd JogoAdvinhacao.ConsoleApp
```
Restaure as dependências
```
dotnet restore
```
Navegue ate a pasta do projeto
```
cd JogoAdvinhacao
```
Execute o projeto
```
dotnet run
```
